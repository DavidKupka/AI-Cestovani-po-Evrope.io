# AI Cestovani-po-Evrope.io

Postup tvorby – Prompty AI
      
      Tento web vznikl díky spolupráci s umělou inteligencí. Zde je stručný přehled kroků:
 


    1. Zadání tématu:<Požadavek na vytvoření vícestránkového webu. "(Vytvoř mi web o 4 strankachv na téma cestování po Evropě)"
    2. Návrh struktury: Rozdělení na úvod, destinace, galerii a kontakt.
    3. Generování obsahu: AI připravila texty a základní HTML/CSS kostru. 
    4. Vylepšení vzhledu: Přidán moderní design – barvy, fonty, galerie s efekty.(vylepsi vzhled stranky)
    5. Dokumentace: Doplněna tato sekce, která vysvětluje postup tvorby.


Používal jsem Microsoft Copilota
Použité technologie
    - HTML5 – struktura webových stránek (nadpisy, sekce, navigace, formuláře)
    - CSS3 – kaskádové styly pro vzhled (barvy, typografie, layout, responzivní design)
    - AI generování textu – obsah a popisy byly připraveny pomocí promptů pro AI
    - AI generování kódu – AI navrhla základní HTML a CSS kostru
    - Obrázky – ilustrační fotografie z [picsum.photos](https://picsum.photos)
    - Responzivní design – CSS media queries pro mobilní zařízení
    - Ikony a efekty – CSS pseudo-elementy (✔, ⚙) a animace při najetí myší
